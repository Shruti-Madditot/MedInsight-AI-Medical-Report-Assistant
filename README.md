# 🤖 MedInsight AI – An Intelligent Medical Report Assistant

## 📌 Overview

**MedInsight AI** is an Artificial Intelligence-powered healthcare assistant that uses **Retrieval-Augmented Generation (RAG)** and **Agentic AI** to analyze medical reports and answer user queries intelligently.

The project combines **Large Language Models (LLMs), Vector Databases, Semantic Search, Embeddings, and AI Agents** to create an interactive system capable of understanding medical reports and providing meaningful responses.

This project demonstrates the practical application of modern Generative AI technologies in solving real-world healthcare problems.

---

# 🎯 Objective

The primary objective of this project is to design and develop an **AI-powered application** that enables users to interact with medical reports using natural language.

The project showcases how multiple AI technologies can work together to build an intelligent assistant capable of understanding documents, retrieving relevant information, selecting appropriate AI tools, and generating accurate responses.

# ▶️ How to Run

1. Clone this repository.
2. Install dependencies using:

```bash
pip install -r requirements.txt
```

3. Open the Jupyter Notebook.
4. Add your Groq API Key.
5. Run all notebook cells.
6. Upload a medical report.
7. Interact with the AI assistant.
