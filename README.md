# 🩺 MedInsight AI - Medical Report Assistant

An AI-powered Medical Report Assistant that helps users understand medical reports using **Retrieval-Augmented Generation (RAG)**, **Hugging Face Embeddings**, **FAISS Vector Database**, **Groq Llama 3.3**, and an **Agentic AI workflow**.

---

## 📌 Project Overview

Understanding medical reports can be difficult for many patients. This project simplifies medical report analysis by allowing users to upload a PDF report and ask questions in natural language.

The system retrieves relevant medical information from the uploaded report and generates context-aware answers using a Large Language Model (LLM).

---

## 🚀 How to Run

1. Clone the repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Open the Jupyter Notebook.

4. Add your Groq API Key.

5. Run all notebook cells.

6. Upload a medical report PDF.

7. Ask questions about the report.
